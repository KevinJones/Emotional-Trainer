The following files are in this directory:

www - contains source code which should be uploaded to a web server.
sql - contains a compressed SQL database, which can be imported through PhpMyAdmin.
docs - contains documents explaining the system in greater detail:
	Release Notes
	User Manual
	Installation Manual
	research-related documents, such as a summary of stock image survey the team created
	documents of historical interest, such as the original team's project post-mortem
license.txt - contains acknowledgement and information about the GPL license under which Emotional Trainer has been released.
